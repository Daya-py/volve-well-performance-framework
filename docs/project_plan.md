# Project Plan

## Working Title

**An Integrated Framework for Multi-Parameter Well Performance Evaluation Using Routine Production Data: A Case Study of the Volve Oil Field**

---

## Problem Statement

Routine production and operational data are widely used in production surveillance and well performance evaluation. However, these data are often analyzed independently or within narrowly defined workflows. This study investigates whether an integrated methodology can provide a more comprehensive approach to well performance evaluation using routine production data.

---

## Aim

To develop and assess an integrated framework for multi-parameter well performance evaluation using routine production data from the Volve Oil Field.

---

## Research Objectives

1. Characterize production and operational behaviour using routine production data.
2. Investigate relationships between operational parameters and production response.
3. Develop an integrated framework for well performance evaluation.
4. Assess the applicability of the proposed framework using the Volve dataset.

## Success Criteria

The study will be considered successful if it:

* Establishes a well-supported research gap.
* Develops a technically sound and reproducible methodology.
* Produces engineering insights from routine production data.
* Demonstrates the applicability of the proposed framework using the Volve dataset.
